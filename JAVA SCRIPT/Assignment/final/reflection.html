Explain the concept of Responsive Design. What does the term mean?
How does responsive design relate to web technologies?
What did you learn about building web apps for different devices through your research?

Reactive web design is a web development approach that dynamically changes the website according to the size and direction of the device's screen (phone, PC, etc.) used when the user views the website. In the broad sense of a responsive website, it is one website corresponding to various characteristics of various devices, which also means that the layout changes depending on the size of the browser (screen size) in real-time. These responsive web designs are usually defined on mobile platforms such as smartphones and tablets using the principle of "mobile-first" and then extended to larger screens.

Through this course, I learned object-based script programming language JavaScript and Node.js, HTML, and CSS. JavaScript causes web pages to be dynamically changed like a live web. In addition, JavaScript runs on a web browser to prevent the overload of servers and is included in HTML documents, making it easy and fast to use. Node.js is a program that enables JavaScript to be used on servers and has the advantage of quickly applying server technology using JavaScript without acquiring a new language. HTML is the most basic markup language for creating web pages and is responsible for the content and structure of web pages. CSS is a stylesheet language used to style documents such as HTML, and although styles can be specified in HTML, CSS is used because it has various advantages such as reusability and improved management.
